# Dropwizard + Gradle = &hearts;

Minimal example of getting Dropwizard going with Gradle.

# Build

Run `./gradlew clean distInall`. The resulting JAR will be saved as `./build/libs/dropwizard-gradle-<<version>>-standalone.jar`.
